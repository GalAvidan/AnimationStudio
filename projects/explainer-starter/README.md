# Explainer Starter

This is the first Remotion proof project for AnimationStudio. It implements `../../specs/explainer-starter_spec.md` as a 10-second animation.

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm run dev
```

**Render video**

```console
npx remotion render src/index.ts ExplainerStarter ../../output/explainer-starter.mp4
```

**Upgrade Remotion**

```console
npx remotion upgrade
```

# Script: Character Pilot — general

## Working Title

Character Pilot — Two Friends, One Question

## Audience

General. Demo audience for the AnimationStudio character-rig pilot.

## Target Runtime

~15 seconds

## Tone

Warm, simple, a little anxious then resolving to reassured.

---

## Script

Opening — set the stage.

[VISUAL: hero alone in a quiet dark room, calmly present]

[SPEAKER: hero]
[SYNC: hero-question]
Why is everything so quiet?

[VISUAL: friend fades in beside hero, glancing back]

[TRANSITION: fade 400ms]

[SPEAKER: friend]
It's okay — I'm right here.

[SPEAKER: hero]
Oh. Then it's not so quiet after all.

[VISUAL: both characters smile and settle]

# Anti-Goals (brief)

Five bullets. Load this at session start. See `anti-goals.md` for full rationale.

- **Not a design app** — not Photoshop, PowerPoint, or a broad animation editor UI.
- **Spec step is mandatory** — no building directly from a prompt without a reviewed spec.
- **No hidden prompts** — all scripts and specs stay human-readable and editable.
- **No root-level output** — all work stays under `{projects}/<name>/`; root `scripts/`, `specs/`, `assets/`, `output/` do not exist.
- **Not a general tool** — if a request doesn't map to a task in `tasks-index.md`, ask before building.

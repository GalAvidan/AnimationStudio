# Skill: Visual Clarity for Explainers

## Purpose

Make animations easier to understand, not merely more decorative.

## Rules

- Show one main idea at a time.
- Use motion to guide attention.
- Keep text short and readable.
- Prefer diagrams, transformations, and spatial relationships over decorative effects.
- Avoid visual clutter around important explanation moments.
- Use contrast and pacing to mark key beats.

## Check

After building or revising, ask: would a viewer understand the point if the animation were muted?

# Task: Refresh Map

## Use When

The folder structure, projects, specs, or assets have changed and the context map needs updating.

## Load

- `agent-context/map/folders.md`
- `agent-context/map/workflow.md`

## Steps

1. Inspect the repository folders.
2. Update folder and workflow maps if new stable locations exist.
3. Add lightweight `derived_from` notes when useful.
4. Do not change intent files without explaining the proposed change.

## Output

Updated map files that reflect the current repository shape.
